v? – What's new

  • Longer session lifetime after login
    Sessions now stay active for a longer period after entering the password, so you are not logged out as quickly.
  • Show password toggle
    A toggle button on password fields lets you reveal or hide the password while typing.
  • Correct link resolution after password entry
    Links are now resolved correctly after the password has been entered, so you are no longer redirected to the home page instead of the intended destination.
  • Payment option hidden when no payment link is configured
    If neither a Wero nor a PayPal link has been provided, the payment option is no longer shown to other users.
  • Open amounts visible to everyone
    The outstanding balance for an order is now displayed for all participants, not just the person who created the order.
